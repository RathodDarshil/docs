The Q1 2025 release delivers significant platform improvements, new SDK capabilities, and enhanced tracking features to elevate your marketing analytics experience.

## Major Features

### Custom Event Tracking

Take control of your conversion funnel with enhanced event tracking:

- **Custom User Events**: Define up to 5 additional user events for your unique conversion path
- **High Volume Support**: Track up to 10 million custom events every month at no additional cost
- **Enhanced User Identification**: Added tracking of 4 new user-level IDs - GAID, GCLID, IDFA, and IDVA

### Campaign Sharing Capabilities

Easily share campaign performance with stakeholders:

- **Targeted Sharing**: Share specific campaign stats with marketing partners or team members via email
- **Streamlined Access**: Recipients get direct access to only the campaign data you choose to share
- **Simple Workflow**: Access the share button from any campaign in the top right corner of the dashboard

### New Data Integrations

Connect your marketing data across your analytics stack:

- **Amplitude Integration**: Seamless data flow between Linkrunner and Amplitude
- **PostHog Integration**: Easily connect your Linkrunner attribution data with PostHog analytics
- **Enhanced API/Webhook Data**: Added Meta campaign level breakdown data including Ad Campaign, Ad Group, and Ad Asset (Name/ID)

### Jio Ads Network Integration

Expanded our ad network capabilities with a new integration:

- **Rapid Implementation**: Built and implemented the Jio Ads Network integration system within two weeks
- **Scalable Architecture**: Created a flexible framework that can be extended to other ad networks (such as Zepto, PhonePe ads)
- **Enterprise-Ready**: Successfully deployed for production use with comprehensive tracking and attribution

## Infrastructure Improvements

### Server Migration & Performance Enhancements

- **Zero-Downtime Migration**: Successfully migrated to new AWS infrastructure with no service interruption
- **Performance Boost**: New infrastructure delivers 12x faster load times, especially for high-volume campaigns
- **Scalability Improvements**: Enhanced system architecture to handle growing data volumes

## Bug Fixes and Improvements

- **Open-in-App Browser Fix**: Resolved tracking issues for open-in-app browsers, improving user experience and achieving 95% user attribution
- **Click Validation**: Implemented improved fraud detection and click validation systems
- **Date/Time Filter Accuracy**: Fixed attribution window calculation issues for precise reporting of downloads and onboards
- **Performance Optimization**: Resolved loading speed issues for high-volume campaigns (100K+ clicks/installs)

## SDK Updates

**Latest Version: LR 1.1.1**

- **Expo Support**: Added new SDK support for applications built with Expo
- **Function Placement Guide**: Enhanced documentation to simplify implementation
- **New Method Structure**: Introduced more specific methods to replace the deprecated `.trigger` method:
  - `linkrunner.signup`
  - `linkrunner.setUserData`
  - `linkrunner.triggerDeeplink`
- **Developer Resources**: Updated documentation and schemas for API/webhook integration

## Documentation Updates

- Launched our official SDK guide for easy integration at [docs.linkrunner.io](https://docs.linkrunner.io)
- Added comprehensive API and webhook documentation with schema details
